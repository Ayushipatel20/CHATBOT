# 🤖 AI Chatbot

An interactive **AI Chatbot** built using **Python, Flask, HTML, CSS, and JavaScript**. The chatbot provides real-time conversations through a modern and responsive web interface. It can answer user queries and perform utility tasks such as weather updates, calculations, date & time, quizzes, and more.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Ayushipatel20/CHATBOT.git
```

### Navigate to the project

```bash
cd CHATBOT
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```

## 🧠 How It Works

1. User enters a message through the chatbot interface.
2. The frontend sends the request to the Flask backend.
3. The chatbot processes the query.
4. A relevant response is generated.
5. The reply is displayed instantly in the chat window.
